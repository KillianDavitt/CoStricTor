* calculating p or q from epsilon

$$p = \frac{q}{(1-q)e^epsilon + q}$$

$$q = \frac{p e^epsilon}{1-p+pe^epsilon}$$

#+begin_src python export:all

        return (q)/((1-q)*(math.e**ep + q))
